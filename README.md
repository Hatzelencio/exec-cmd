# exec-cmd package

## Bash portable

Ejecuta comandos desde un texto seleccionado.

> Instalación

```bash
apm install exec-cmd
```

--------------------------------------------------------------------------------

> ¿Cómo usarlo?

1. Selecciona el texto deseado a ejecutar
2. Utiliza el atajo `alt` + `r`
3. ¡Disfruta! :v
